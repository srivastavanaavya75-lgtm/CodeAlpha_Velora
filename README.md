# Velora Music Player

A modern glassmorphism-inspired music player built using HTML, CSS, and JavaScript.

## Features

- Play / Pause Controls
- Next / Previous Track Navigation
- Search Functionality
- Playlist Management
- Progress Bar
- Volume Controls
- Shuffle & Repeat Modes
- Responsive Design
- Glassmorphism UI

## Tech Stack

- HTML5
- CSS3
- JavaScript
