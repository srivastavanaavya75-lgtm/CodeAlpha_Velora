# 🎵 Velora Music Player

A modern glassmorphism-inspired music player built using HTML, CSS, and JavaScript. Velora delivers a smooth and visually appealing music experience with a lavender-blue aesthetic, responsive design, and interactive playback controls.

## 🌐 Live Demo

🔗 https://velora-music-player.netlify.app

## 📂 GitHub Repository

🔗 https://github.com/srivastavanaavya75-lgtm/CodeAlpha_Velora

---

## ✨ Features

- ▶️ Play / Pause Controls
- ⏭️ Next Track Navigation
- ⏮️ Previous Track Navigation
- 🔀 Shuffle Mode
- 🔁 Repeat Mode
- 🎚️ Interactive Progress Bar
- 🔊 Volume Control
- 🔍 Real-Time Song Search
- ❤️ Like / Favorite Songs
- 🎨 Glassmorphism UI Design
- 📱 Fully Responsive Layout
- 🎵 Dynamic Playlist Management
- 🎼 Album Cover Carousel

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

---

## 🎨 Design Highlights

Velora follows a premium lavender-blue glassmorphism design language featuring:

- Frosted glass UI components
- Smooth gradients
- Modern music card layouts
- Interactive hover effects
- Responsive mobile-friendly design
- Elegant typography and spacing

## 🚀 How to Run Locally

1. Clone the repository

```bash
git clone https://github.com/srivastavanaavya75-lgtm/CodeAlpha_Velora.git
```

2. Open the project folder

```bash
cd CodeAlpha_Velora
```

3. Run the project

Simply open:

```text
index.html
```

in your browser.
